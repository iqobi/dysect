# DYSECT — Vacuum The Darkness

A scary but relaxing 3D apartment insect vacuuming game built with Three.js.

## How to Run

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)

### Install Dependencies

```bash
npm install
```

### Start the Dev Server

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:5173`).

### Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look |
| Left Click | Vacuum |
| F | Flashlight |
| E | Interact |

## Build

```bash
npm run build
npm run preview   # preview the production build
```
